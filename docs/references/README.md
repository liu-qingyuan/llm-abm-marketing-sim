# References

本目录保存外部资料整理、研究先验、数据集说明和不可执行参考。

## 使用规则

- 写这里：研究表格整理、数据口径参考、外部方法说明、只读背景资料。
- 不写这里：实现规格、验收标准、架构决策、当前任务状态。
- Reference 可以被 PRD、Architecture Note 或 ADR 引用，但自身不代表已实现状态。
