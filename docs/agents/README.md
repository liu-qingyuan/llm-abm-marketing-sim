# Agent Skills

本目录保存工程 skills 的稳定工作约定。

## 当前约定

- [Issue tracker](issue-tracker.md)：本仓库使用 GitHub Issues，外部 PR 也作为 triage 请求入口。
- [Triage labels](triage-labels.md)：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。
- [Domain docs](domain.md)：本仓库是单上下文仓库；如存在根目录 `CONTEXT.md` 和 `docs/adr/`，工程 skills 应按需读取。

这些文件只记录长期稳定约定。具体 PRD、issue plan、实现状态和验证结果应放在对应职责目录或 GitHub issues 中。
