## 2. 各价值维度解释

**认知价值**：锦江秸秆产品为消费者提供新奇体验、启发或教育其了解可持续发展，并满足其好奇心的潜力。

示例：“我以前从不知道秸秆竟然有这么高的价值。”

**环境价值**：消费者基于产品与其可持续发展期望、环境关切以及更广泛的可持续发展目标之间的契合程度，对产品价值所作出的评价。

**功能价值**：价格，即经济价值；以及质量，即实用性利益。

**健康价值**：消费者对健康和身心福祉的感知。

示例：“你不担心这些秸秆是用粪肥处理过的吗？”

**情感价值**：消费者产生的积极情感反应。

**社会价值**：使用与特定社会、人口、经济或文化群体相关的产品或服务时，消费者自我形象的提升以及所感知到的社会认可。

示例：“作为锦江旗下酒店的一名员工，我感到自豪！”

## 4. 后续使用说明

- 类别分配概率可作为生成 `latent_class` 的外部先验分布。
- 选择模型组成部分的系数可作为每个 class 的价值权重。
- 表 11 更适合作为各 class 内部的成员画像分布，即 `P(profile label | class)`。
- 酒店档次指最近一次入住锦江酒店的酒店档次。
- 旅行目的指最近一次入住锦江酒店的旅行目的。
- 价值维度的适用语境是锦江酒店秸秆产品或相关绿色服务，不应泛化为用户整体人格或长期消费观。
- 性别、年龄、教育、收入等画像字段在本项目中应作为仿真实验标签和分组分析标签使用，不代表 Douyin 用户真实身份。
