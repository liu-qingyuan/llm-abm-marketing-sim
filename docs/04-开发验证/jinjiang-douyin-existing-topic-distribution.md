## 7. 标题包含情况：标题里是不是必须有 `#锦江酒店`？

针对 related unbounded run 里的 `1614016211862532` / `锦江酒店` 话题页：


| 口径                                          | 去重视频数 | 说明                             |
| ------------------------------------------- | ----- | ------------------------------ |
| 来自 `#锦江酒店` 话题页的视频                           | 696   | 以 source challenge/page cid 为准 |
| caption/desc/title 含 `#锦江酒店`                | 695   | 严格标题文本 hashtag 口径              |
| caption/desc/title 含 `锦江酒店`                 | 695   | 标题文本包含品牌词口径，不要求 `#`            |
| hashtags/cha_list/text_extra 含 exact `锦江酒店` | 696   | 结构化标签字段口径                      |


解释：

- “来自 `#锦江酒店` 话题页”不等于“标题文本必须出现 `#锦江酒店`”。
- TikHub/Douyin 的话题页接口可能返回标题为空、标签字段缺失、或带有混排/推荐逻辑的视频。
- 如果用户要“标题必须含 `#锦江酒店`”，需要在话题页结果上再做严格标题过滤。
- 如果用户要“hashtags 字段含 exact `锦江酒店`”，这是另一个结构化标签字段口径。
