# 05-周报

本目录保存面向老师快速阅读的周期小结。周报属于 reports archive，用于概括某个统计周期的核心成果和下一步，不作为当前架构、实现规格或数据集合同的 canonical 入口。

## 使用规则

- 写这里：标题、统计周期/口径更新时间、核心成果、下一步。
- 不写这里：PRD、验收标准、架构决策、研究先验、数据集明细或用户明细。
- 周报可以引用 canonical 文档，但不替代 `../prds/`、`../architecture/`、`../references/`、`../adr/` 或具体 run/report 的状态说明。
