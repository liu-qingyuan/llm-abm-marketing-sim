# 本周工作进展报告（2026.06.30-2026.07.06）：锦江酒店 Douyin 用户潜在属性建模与仿真实验准备

## 小结

> 本周统计周期：`2026.06.30-2026.07.06`；核心数据口径更新时间：`2026.07.05`。

本周主要完成了锦江酒店 Douyin 用户数据从“可观测画像指标”到“可用于 ABM 仿真实验分组”的扩展。前期已经完成 36,400 个用户的最终数据集整理，本周进一步把文献中的用户消费偏好类型转化为可复现的实验标签，使后续可以比较不同类型用户对营销内容的互动差异。核心成果包括：

- 将 36,400 个锦江 Douyin 用户划分为三类消费偏好用户群体，用于后续仿真实验中的分组比较；
- 三类用户群体数量分别为 `15,616`、`15,070`、`5,714`，与预设比例完全一致，说明本次实验标签分配过程可复现、可审计；
- 为每个用户补充了六类消费价值偏好信息：知识/新奇价值、环保价值、功能价值、健康价值、情感价值和社交价值；
- 为每个用户补充了便于解释实验结果的画像标签：酒店档次偏好、出行目的、性别、年龄、教育程度和月收入区间；
- 明确这些新增标签是“仿真实验标签”，用于模拟不同消费偏好人群的传播行为，不代表对 Douyin 用户真实身份或真实人口属性的判断；
- 保留原始 final dataset 不变，另外生成一个新的实验标签派生数据集，便于和原始数据口径对照；
- 完成 36,400 个用户的本地验收：三张核心用户表仍均为 `36,400` 行，用户集合完全一致，每个用户都成功获得实验标签；
- 将营销内容也整理为可比较的价值维度，使后续可以分析“强调环保/健康/社交等价值的内容”更容易影响哪类用户；
- 完成按用户类型、消费价值偏好和出行画像进行分组统计的准备，为后续比较不同群体的传播效果打基础；
- 同步整理项目文档入口和说明文档，使数据来源、实验标签含义、验收结果和后续使用方式更加清楚。

## 当前口径说明

本周新增内容不是重新采集数据，也不是对真实用户做身份识别，而是在已有锦江 Douyin 数据集上增加一层用于仿真实验的用户分组标签。当前口径如下：

| 项目 | 当前说明 |
|---|---|
| 原始数据 | 继续使用 36,400 个锦江 Douyin 用户的 final dataset，不覆盖原数据 |
| 新增实验标签 | 三类消费偏好用户群体、六类消费价值偏好、酒店/出行/人口画像解释标签 |
| 标签用途 | 用于 ABM 营销传播仿真实验中的用户分组、内容匹配和结果解释 |
| 不是 | 不是真实人口识别，不是真实心理画像，不是第三方认证标签 |
| 验收结果 | 36,400 个用户全部完成标签生成，目标比例与实际分配一致 |
| 展示方式 | 后续报告以群体统计、方法说明和不同用户类型的对比结果为主 |

因此，本周工作可以概括为：**在已完成的锦江酒店 Douyin final dataset 基础上，构建了一套可复现、可审计的用户消费偏好实验标签，并完成本地验收和文档整理。下一步可以开始设计具体仿真实验，例如比较环保型、健康型、社交型营销内容在不同用户群体中的传播差异。**
