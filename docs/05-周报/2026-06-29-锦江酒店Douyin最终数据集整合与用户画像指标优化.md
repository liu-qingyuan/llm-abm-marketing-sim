## 当前指标口径

当前用户画像指标采用 `ln(x+1) + P95` 分位数归一化方法。三类核心 proxy 的解释边界如下：

| 指标 | 字段 | 解释边界 |
|---|---|---|
| Activity | `activity_score` | 用户在本研究语境中的内容生产、评论、回复活跃度代理 |
| Global Influence | `global_influence_score` | 基于 `follower_count` 的平台潜在覆盖力代理 |
| Local Influence | `local_influence_score` | 基于评论网络连接与评论获赞的锦江语境局部影响力代理 |

因此，本周工作可以概括为：**完成了锦江酒店 Douyin 最终研究数据集的用户画像补齐、数据整合、旧产物清理和 profile 指标方法升级，使该数据集已经可以作为后续 LLM-ABM 社交传播仿真的主要输入；下一步应围绕指标 reference 继续扩展文献依据，并开始设计 ABM 初始化、网络扩散实验和论文方法段落。**
